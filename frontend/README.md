# Quizory Frontend (independent project)

Ovo je zaseban **React + TypeScript** projekat i može se razvijati nezavisno od backend repoa/projekta.

## Pokretanje
```bash
cd frontend
npm install
npm run dev
```

## Backend URL
Frontend koristi `VITE_API_URL`.

Primer:
```bash
VITE_API_URL=http://localhost:5000/api npm run dev
```

Ako nije podešeno, koristi `http://localhost:5000/api`.
